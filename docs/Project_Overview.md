# Project Overview

## Title
VLSI-Based Digital Clock with Alarm Functionality

## Objective
Design and verify a digital clock using Verilog HDL that displays time in 24-hour format and generates an alarm when the current time matches a user-defined alarm time.

## Features
- Clock Divider
- Seconds Counter
- Minutes Counter
- Hours Counter
- Alarm Controller
- RTL Design
- Verification Testbench
- GTKWave Analysis

## Applications
- Digital Watches
- Embedded Systems
- FPGA Controllers
- Industrial Timing Systems
